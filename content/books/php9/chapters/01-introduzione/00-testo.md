# Introduzione

PHP è uno dei linguaggi più diffusi per lo sviluppo web lato server. Nato
come strumento per generare pagine dinamiche, negli anni si è evoluto in
un linguaggio maturo, con un sistema di tipi via via più espressivo e un
ecosistema di framework e strumenti solido.

Questo libro si concentra sul PHP moderno: la sintassi di base e il
sistema dei tipi che permette di scrivere codice più sicuro e facile da
mantenere.
