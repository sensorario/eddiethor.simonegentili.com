# Introduzione

Git è il sistema di controllo di versione più diffuso al mondo. Nato nel 2005
per le esigenze dello sviluppo del kernel Linux, oggi è lo strumento di base
per praticamente ogni progetto software, piccolo o grande che sia.

> "Non sono un buon programmatore. Sono solo un bravo programmatore con
> ottime abitudini." — variazione su un tema caro a molti sviluppatori Git.

In questo libro vedremo i comandi essenziali per iniziare a usare Git in
modo produttivo, senza perdersi nei dettagli più avanzati.
