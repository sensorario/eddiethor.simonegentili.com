# Comandi di base

Il ciclo di lavoro quotidiano con Git ruota attorno a pochi comandi
fondamentali: inizializzare un repository, salvare le modifiche e
consultarne la storia.

```bash
git init
git add file.txt
git commit -m "Primo commit"
git log --oneline
```

Una volta presi confidenza con questi quattro comandi, il resto del
flusso di lavoro (branch, merge, remote) diventa molto più naturale.
