### D: Che cosa è una funzione in TypeScript?
Una funzione, in TypeScript, è un blocco di codice riutilizzabile e serve per evitare ripetizioni di codice.

### D: Perché è utile specificare il tipo di ritorno di una funzione?
Perché rende il codice più robusto.

### D: Perché conviene tipizzare i parametri di una funzione?
Per evitare che vengano passati valori non consentiti o addirittura tipi sbagliati.

### D: Come si indica il tipo di ritorno in una funzione?
Ad esempio se il tipo di ritorno è stringa, si può usare la seguente sintassi.

```typescript
function fooBar() : string {
  return "some string";
}
```

### D: È obbligatorio dichiarare il tipo di ritorno di una funzione?
No ma farlo rende il codice più leggibile e solido.
