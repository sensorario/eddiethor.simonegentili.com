# Introduzione

TypeScript è un superset di JavaScript che aggiunge un sistema di tipi
statico. Il codice TypeScript viene compilato in JavaScript semplice, ma
lungo il percorso il compilatore individua molti errori prima ancora
che il codice venga eseguito.

Questo libro parte dall'introduzione al sistema dei tipi fino ai tipi di
base che si usano quotidianamente scrivendo TypeScript.
