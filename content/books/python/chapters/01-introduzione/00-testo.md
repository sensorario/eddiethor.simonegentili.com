# Introduzione

Python è un linguaggio di programmazione general-purpose noto per la sua
sintassi leggibile e per l'ampio ecosistema di librerie, che spaziano
dallo scripting all'analisi dati fino allo sviluppo web.

Questo libro accompagna chi muove i primi passi con Python, partendo
dalla sintassi essenziale fino alle strutture dati che si usano ogni
giorno.
