# Strutture dati

Python offre alcune strutture dati fondamentali già pronte all'uso:
liste, tuple, dizionari e insiemi.

```python
numeri = [1, 2, 3]
punto = (10, 20)
persona = {"nome": "Ada", "eta": 30}
unici = {1, 2, 3}
```

Le liste sono mutabili e ordinate, le tuple sono immutabili, i
dizionari associano chiavi a valori e gli insiemi garantiscono
l'unicità degli elementi. Scegliere la struttura giusta rende il codice
più chiaro ed efficiente.
